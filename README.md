# YouTube Clone

A modern YouTube-inspired web application that allows users to browse, search, and watch videos with a clean and responsive user interface.

---

## Features

* User-friendly homepage
* Video search functionality
* Responsive design for mobile and desktop
* Video player integration
* Sidebar navigation
* Trending and recommended videos section
* Channel details display
* Like, comment, and subscribe UI

---

## Tech Stack

### Frontend

* HTML
* CSS

---

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/youtube-clone.git
```

### Navigate to the project folder

```bash
cd youtube-clone
```

### Open the project

Simply open the `index.html` file in your browser.

```bash
index.html
```
